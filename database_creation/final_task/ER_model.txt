Основные сущности:

User — пользователи системы.
- id (PK)
- email (логин)
- birth_date (дата рождения)
- birth_time (время рождения)

ForecastType — типы прогнозов.
- id (PK)
- name (например, "знак зодиака", "китайский", "цыганский")
- monthly_price (стоимость подписки за месяц)

Subscription — подписки пользователей.
- id (PK)
- user_id (FK, ссылка на User)
- forecast_type_id (FK, ссылка на ForecastType)
- start_date (дата начала подписки)
- duration_months (длительность подписки в месяцах)

Связи:

Пользователь может иметь несколько подписок, каждая из которых относится к конкретному типу прогноза.
У каждого типа прогноза фиксированная месячная стоимость.
